## 🧠 AI Toolkit 到底解决了什么问题？

在 AI 开发中，我们通常要做：

1. 找模型（OpenAI / Anthropic / 本地模型）
2. 测试 Prompt
3. 构建 Agent
4. 调试工作流
5. 评估效果
6. 部署上线

**问题是**：这些工具往往是分散的。

**AI Toolkit 的核心价值**：把这些全部整合进 VS Code，一个工具搞定。

## ✨ 什么是 AI Toolkit

AI Toolkit 就是一个装在 VS Code 里的 AI Agent 开发工具。用它来构建、测试和优化 AI Agent。

它内置了 Microsoft Foundry 扩展，让你无需离开 VS Code，就可以直接访问 Microsoft Foundry 资源——例如部署模型、管理 Agent 等。

> 📸 图片建议：VS Code 中 AI Toolkit 整体界面截图

## 🔍 核心能力一：模型发现与评估（Model Catalog）

AI Toolkit 提供统一的模型入口：

- Microsoft Foundry
- Foundry Local
- OpenAI / Anthropic / Google
- GitHub Models
- 本地模型（ONNX / Ollama）

你可以直接在 VS Code 里：

- 👉 浏览模型
- 👉 对比模型
- 👉 快速测试效果

> 📸 图片建议：Model Catalog 界面（多模型列表 + 对比）

## 💬 核心能力二：Playground（模型实验环境）

Playground 是一个交互式聊天环境。你可以：

- 测试 Prompt
- 调整参数（temperature / top_p）
- 使用多模态输入（图片 / 文件）
- 实时查看模型响应

相当于一个内置的 ChatGPT + 调试工具。

## 🤖 核心能力三：AI Agent 构建（重点🔥）

### 🖱️ 无代码方式：Agent Builder

- 用自然语言生成 Prompt

- 可视化 Agent 工作流
